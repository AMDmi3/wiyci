The World Is Your CI

F/OSS projects get packaged in distributions and repositories, wiyci collects package build logs from these and present in the form of CI builds.

This can be useful to upstreams (gives an overview of downstream builders, and much wider variety of different build environments that any traditional CI would provide), contributors (easy way to find a project which need improvement), compiler developers (find cases where warnings are triggered, introduce new warning classes), platform developers (to find places to improve specific to their platform of interest).

wiyci is currently in Alpha, so service capabilities are limited. But we're growing!

wiyci currently supports build logs from Alpine, Fedora and FreeBSD. This already covers 2 different kernels, 2 different C libraries, and 8 hardware arhitectures. Suggest more sources!

wiyci currently parses C/C++ compiler warnings and CTest/Google Test/PyTest test failures. Suggest more useful facts to parse!

wiyci currently only supports a few dozen most popular projects. Suggest a project you're really interested in!
